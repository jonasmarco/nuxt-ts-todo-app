# Todo App Minimal

Projeto de lista de tarefas minimalista utilizando Nuxt com TypeScript.

## Stack Utilizada

- Nuxt 3
- Vue.js
- TypeScript
- Node.js

## Funcionalidades

- Adicionar novas tarefas
- Marcar tarefas como concluídas/incompletas
- Editar e remover tarefas
- Filtragem por estado (todas, ativas, concluídas)

## Comandos Básicos

Confira os scripts no arquivo package.json para instalar dependências, iniciar o desenvolvimento e gerar builds.
